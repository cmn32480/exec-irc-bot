IRCiv

real-time play-by-irc civilization-building game

Game objective:

To build a civilization, make money, fend off enemies to maintain borders, expand, etc (similar to other civilization-branded games).

IRC commands:

Messages should be sent privately to the irc bot nick (to avoid password visibility on a public channel), typing "/msg servernick password command", although it's probably easier use a query window (private message to server nick) to send commands, once opened just typing "password command" each time.

general syntax: password action args

action: insert|update|delete

Specific actions should operate on the currently highlighted unit by default (to eliminate need to enter unit identifiers). After each action, the program should automatically go to the next available unit and output a description.

Currently supported commands:

/msg exec civ <password> insert player <email>

Map/world design:

Not sure if a Trade Wars 2002 / Spiral Arm type mapping approach should be taken, or a 2D tile-based approach. The world should be easily scalable though.
